# Pre-Start Up
* Download node.js from node official website

* Open terminal, npm install --global expo-cli
(expo CLI allow create and run RN easily)

## Way 1:
- Download Android studio code (Or use own mobile to run the apps)
- run the android device emulator from Android Studio Code

## Way 2
Mobile phone download 'expo go' apps from play store
Use the scanner inside 'expo go' to run the apps

## Way 3
Or can run in web (type w in the terminal)

# Start Up Manual

## AI
* cd AI
* python main.py

## Front End
* cd UnLost
* npx expo start
